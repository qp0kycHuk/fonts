# Fonts packages

## Установка

```bash
npm install @qpokychuk/fonts --save
```

## Основа использования

Импортируйте нужный шрифт в js или css

```js
import '@qpokychuk/fonts/inter/index.css';
```

```css
@import '@qpokychuk/fonts/inter/index.css';
```

## Доступные шрифты:

inter - `font-family: 'Inter'`
